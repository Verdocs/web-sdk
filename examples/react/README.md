# Verdocs Web SDK — React Examples

Example applications for integrating the Verdocs React SDK (`@verdocs/web-sdk-react`).

## Available examples

| Example | Description |
|---------|-------------|
| [verdocs-build-example](./verdocs-build-example/) | Template builder embed (`VerdocsBuild`), white-label field styling, recipient sequence/order |

### Quick start

```bash
cd verdocs-build-example
cp .env.example .env
npm install
npm run dev
```

See each example’s README for credentials, workflow details, and styling guidance.
